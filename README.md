# 実験する数学 — 夏期講習会III

城北中学校・高等学校の中学3年・高校1年生向けに、プログラミングを使って数学を学ぶ夏期講習会の教材リポジトリです。メイン言語はJuliaですが、同じ内容をPython・Rでも体験できる版を用意し、言語ごとの書き方の違いも学べるようにしています。

## 開催概要

- **日程**: 2026年8月24日(月)〜8月28日(金) の5日間
- **担当**: 清水団
- **形式**: 昨年と同様、A帯・B帯の2クラス制、各回50分×5日間
- **コンセプト**: 「Juliaは計算するための言語ではなく、『数学を実験するための言語』である」
  - 整数は「探す」
  - 場合の数は「全部作る」
  - 関数は「描く」
  - 確率は「試す」

詳しい構成の検討メモは [intro.md](intro.md) を参照してください。

## カリキュラム(2026年)

Julia版をベースに、Python版・R版でも同じ5日間の内容を体験できます。

| Day | テーマ | 内容 |
| --- | --- | --- |
| 1 | Colabで計算しよう | Google Colab、基本文法、数式入力、for文、if文 |
| 2 | 整数問題をプログラムで考えよう | 約数・素数判定・エラトステネスの篩・最大公約数、コラッツ予想など |
| 3 | 場合の数・組合せを実験しよう | 順列・組合せの全列挙、パスカルの三角形、シェルピンスキーガスケット |
| 4 | 関数のグラフを描いて最大・最小を求めよう | 関数、グラフ、パラメータ、数値実験 |
| 5 | 確率とシミュレーション | コイン、サイコロ、誕生日問題、モンティ・ホール問題、モンテカルロ法 |

## ディレクトリ構成

```
.
├── intro.md             # カリキュラム構成の検討メモ
├── 2025summer/          # 2025年度の教材(前年実施分、Juliaのみ)
└── 2026summer/          # 2026年度の教材
    ├── julia_dayN_colab.ipynb    # Julia版・各日の講義用ノートブック
    ├── julia_dayN_answers.ipynb  # Julia版・各日の解答例ノートブック
    ├── julia_dayN_slides.pdf/.pptx  # 各日のスライド
    ├── summer_seminar_2026.pdf/.pptx  # 全体概要スライド
    ├── Python/
    │   └── python_dayN_colab.ipynb  # Python版・各日の講義用ノートブック
    └── R/
        └── r_dayN_colab.ipynb       # R版・各日の講義用ノートブック
```

(`N` は 1〜5 の日程番号)

### 2026summer/

- **Julia版**(メイン教材): `julia_dayN_colab.ipynb` / `julia_dayN_answers.ipynb` / スライド一式
- **Python版**(`Python/`): Julia版と同じ5日間の内容をPythonで体験するノートブック。Colabの標準言語なので環境設定は不要。matplotlib・numpy(Day4)、japanize-matplotlibなどを使用
- **R版**(`R/`): Julia版と同じ5日間の内容をRで体験するノートブック。ColabでR実行環境に切り替えて使用(ランタイム → ランタイムのタイプを変更 → R)。グラフの日本語ラベルは文字化けを避けるため英語表記

各言語版のノートブックはGoogle Colabで編集・実行され、GitHub連携で本リポジトリに同期されています。
