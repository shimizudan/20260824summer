# 実験する数学 — Julia 夏期講習会

高校生向けにJuliaを使ってプログラミングと数学を学ぶ夏期講習会の教材リポジトリです。

## 開催概要

- **日程**: 2026年8月24日(月)〜8月28日(金) の5日間
- **形式**: 昨年と同様、A帯・B帯の2クラス制、各回50分×5日間
- **コンセプト**: 「Juliaは計算するための言語ではなく、『数学を実験するための言語』である」
  - 整数は「探す」
  - 場合の数は「全部作る」
  - 関数は「描く」
  - 確率は「試す」

詳しい構成の検討メモは [intro.md](intro.md) を参照してください。

## カリキュラム(2026年)

| Day | テーマ | 内容 |
| --- | --- | --- |
| 1 | Juliaを使って計算しよう | Google Colab、基本文法、数式入力、for文、if文 |
| 2 | 整数問題をプログラムで考えよう | 約数・素数判定・エラトステネスの篩・最大公約数、コラッツ予想など |
| 3 | 場合の数・組合せを実験しよう | 順列・組合せの全列挙、パスカルの三角形、シェルピンスキーガスケット |
| 4 | 関数・グラフ・最大最小 | 関数、グラフ、パラメータ、数値実験 |
| 5 | 確率・シミュレーション | コイン、サイコロ、誕生日問題、モンティ・ホール問題、モンテカルロ法 |
